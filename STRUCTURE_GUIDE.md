# Инструкция по созданию структур для Bulletin Board Mod

## ⚠️ Важно: текущее состояние

В версии **1.3.0** worldgen-структуры **отключены** до тех пор, пока не будет создан NBT файл структуры.

**Команды работают без структур!** Вы можете использовать `/bulletin trigger` и другие команды на любых досках, размещённых вручную.

---

## 📦 Создание NBT файла структуры

### Шаг 1: Постройка структуры
1. Запусти Minecraft с установленным модом
2. Создай постройку доски объявлений (или используй готовую)
3. Выдели область с помощью `/structure save`

### Шаг 2: Экспорт структуры
Используй команду:
```
/structure save bulletin-board:village_bulletin_board <x1> <y1> <z1> <x2> <y2> <z2>
```

Где `<x1> <y1> <z1>` и `<x2> <y2> <z2>` - противоположные углы области.

**Пример:**
```
/structure save bulletin-board:village_bulletin_board 100 64 200 103 67 202
```

### Шаг 3: Размещение файла
После экспорта файл появится в:
```
.minecraft/structures/bulletin-board/village_bulletin_board.nbt
```

Скопируй его в проект:
```
src/main/resources/data/bulletin-board/structures/village_bulletin_board.nbt
```

## 🏗️ Требования к структуре

### Размеры
- **Максимальный размер:** 48x48x48 блоков
- **Рекомендуемый:** 3x3x2 (для одиночной доски) или 5x3x2 (для двойной)

### Позиционирование
- **Точка привязки (pivot):** Нижний левый задний угол структуры
- **Высота:** 0 означает поверхность земли
- **Ориентация:** Доска должна смотреть "лицом" к положительной Z (на юг)

### Примеры структур

#### 1. Простая доска у колодца
```
Размер: 3x3x2
Состав: Доска объявлений + 2 блока земли/камня вокруг
Размещение: Рядом с деревенским колодцем
```

#### 2. Информационный стенд
```
Размер: 5x4x3
Состав: Доска + навес + фонарь
Размещение: В центре деревни
```

#### 3. Доска у дома старосты
```
Размер: 3x2x1
Состав: Только доска объявлений
Размещение: На стене дома старосты
```

## 🔧 Типы деревень для интеграции

Мод поддерживает следующие типы деревень:
- `plains` (равнинная)
- `desert` (пустынная)
- `savanna` (саванна)
- `snowy` (заснеженная)
- `taiga` (таёжная)

Для каждого типа можно создать отдельную структуру:
```
village_bulletin_board_plains.nbt
village_bulletin_board_desert.nbt
village_bulletin_board_savanna.nbt
village_bulletin_board_snowy.nbt
village_bulletin_board_taiga.nbt
```

## 🎲 Тестирование

### Команды для проверки:
1. **Создать структуру рядом:**
   ```
   /structure load bulletin-board:village_bulletin_board ~ ~ ~
   ```

2. **Вызвать событие с запиской:**
   ```
   /bulletin trigger 10
   ```

3. **Разместить записку на конкретной доске:**
   ```
   /bulletin trigger-at <x> <y> <z>
   ```

4. **Проверить записки на доске:**
   ```
   /bulletin list <x> <y> <z>
   ```

5. **Очистить доску:**
   ```
   /bulletin clear <x> <y> <z>
   ```

---

## 📝 Примечания

1. **NBT файл** должен быть размещён ДО запуска игры
2. **Перезапуск мира** требуется после добавления новых структур
3. **Отладка:** Включи `/gamerule showCoordinates true` для точного позиционирования
